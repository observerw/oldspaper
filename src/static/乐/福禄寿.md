# 简介

我从第一次听《用什么把你留住》到写这篇文章专门来夸福禄寿，之前不过相隔三天。之所以这么火急火燎，就是因为这支乐队太牛逼了，对于已经干涸了一段时间的我而言仿佛一股清泉，带给我的惊讶程度不亚于我第一次听到mili时的感觉。
