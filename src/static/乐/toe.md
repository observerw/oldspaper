# 简介

Toe是一支数学摇滚乐队。这表明他们玩的并不广泛，在摇滚庞大的门类中他们只占据了其中一个相对冷门的分支。但就是在数学摇滚这一分支上，他们做到了登峰造极，直到现在我仍然可以说出，toe是我听过的最好的数学摇滚乐队。
